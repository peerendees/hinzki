# Referenz-Logos

## Anweisungen zum Herunterladen

1. Gehe auf https://haushinzki.de/
2. Scrolle zu den Referenzen
3. Rechtsklick auf jedes Logo → "Bild speichern unter"
4. Speichere die Logos hier mit beschreibenden Namen:

## Empfohlene Dateinamen:
- `logo-client-1.png`
- `logo-client-2.png` 
- `logo-client-3.png`
- etc.

## Oder mit wget (falls URLs bekannt):
```bash
# Beispiel:
wget -O logo-client-1.png "https://haushinzki.de/path/to/logo1.png"
```

## Unterstützte Formate:
- PNG (empfohlen für Logos)
- SVG (beste Qualität)
- JPG/JPEG
- WebP
